# Version 1.3

## Additions

- New `logger` in every `Geode` instance for easily logging messages without having to create your own.

### Data Generation

#### `SoundRunnable`
- Creating a sound datagen entry without a subtitle translation doesn't crash the game when running datagen anymore.
- Added `sound(String)`, providing the name of a sound file to be used in this sound event. Format:
  - Located under `assets/NAMESPACE/sounds`.
  - The extension doesn't need to be specified.
